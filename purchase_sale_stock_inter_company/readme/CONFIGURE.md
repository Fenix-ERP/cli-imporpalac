To configure this module, you need to: \#. Go to the menu _Settings \> General
Settings_. \#. Go to the tab _Companies / Inter Company OCA features_, in the group
_Purchase/Sale_, check the option _Sale from purchase_. \#. Then, select the _Warehouse
for Sale Orders_, it is the warehouse that will be used to automatically generate the
sale order in the other company.

1.  If you check the option "Sync picking" (for both companies), validating a picking
    generated by an inter-company purchase/sale will validate the respective picking for
    the other company.
2.  If you check the option "Block manual validation of picking in the destination
    company" is not possible to validate manually the picking in the destination
    company.
